ℹ️ Kako deluje

Branje je odprto za vse — brez prijave.
Interakcija (👍 💬 ⚡) zahteva Nostr profil.
Vsebina teče prek javnih Nostr relayjev. Kaj je Nostr? →

🛒 O tržnici

Ponudbo objavijo domači kmetje in obrtniki sami (NIP-99).
Za oddajo oglasa ali stik potrebuješ Nostr profil.
Portal je oglasna deska, ne trgovina — plačilo je dogovor.
Oglasi in slike so moderirani; sporne prijaviš.

✓ Verificirani ponudniki

Značka ✓ = potrjena istovetnost (NIP-05), npr. kmetija-novak@ig.lokalna-skupnost.si.

🎓 O izobraževanju

Učne ponudbe objavijo izvajalci sami (tečaji, delavnice, inštrukcije, gradiva).
Vsebine so brezplačne ali plačljive — filtriraj po znački.
Portal je imenik ponudb — prijava in plačilo potekata pri izvajalcu.
Za oddajo ponudbe ali prijavo potrebuješ Nostr profil.

🔐 Zakaj Nostr?

Tvoj profil je res tvoj — prenosljiv med aplikacijami, ne ujet v to platformo.
Brez osrednjega lastnika — ni oglasov in sledenja; podatki so odprti.
Tvoje objave in stiki ostanejo tvoji, tudi če zamenjaš aplikacijo.
